SET

Rules

How to win:

The goal of the game is to collect all of the sets in the game as quickly as possible. At the end of the game your score will be calculated based on how many sets you acquired and how long it took you.

What is a set?:

Every card in the game has 4 properties: number, color, shading and shape.

A set is a collection of 3 cards that, *for each property*, is all the same or all different. That means all 3 cards can have the same color (i.e purple) or they must all have different colors (i.e one of each color).

Just click on the cards to select them. When you've selected three the game will let you know if you have found a set with a green flash.

What if there are no sets on the board?

Nope! Until the game is over, there will always be a valid set on the board. The game will automatically deal 3 extra cards if there it is impossible to make a set.
