Ruby Enumerables Cheat Sheet

Ruby Enumerables Cheat Sheet

Most Common Ruby Enumerables:

  • collect/map
  • each
  • inject/reduce
  • select
  • sort
Name Explanation Example Code
Collect/Map Returns a new array based on running whatever is in the block of code over each element of whatever you pass into the method. #collect and #map are two names for this same thing. 
(1..4).map {|x| x* x} #=> [1, 4, 9, 16]
(1..4).collect {5} #=> [5, 5, 5, 5]
Each Each is technically not a part of the enumerables mixin. Instead it is the method required for a class to be able to use the enumerables method. #each calls the given block of code once for each element in the object it's called on (usually an array or hash) passing in that element as a parameter. 
a = ["a", "b", "c"]
a.each {|x| print x, " -- "} #=> a -- b -- c
Inject/Reduce Inject combines all the elements of the object passed into it by applying an operation specified by the block of code. Inject starts with an initial value, applies the block of code with each element and keeps track of the total by replaceing an accumulator value. At the end it returns the accumulator value. By default the initial value is the first element of the object you pass in, but you can also set that initial value. Inject and reduce are two names for this thing. 
# Sum some numbers
(1..4).reduce(:+) #=> 10
(1..4).inject {|sum, n| sum + n} #=> 10
# Multiply numbers
(5..8).inject {|product, n| product * n} #=> 1680
# Multiply numbers with initial value set to 10.
(5..8).inject(10) {|product, n| product * n} #=> 16800
Select Select returns a new array containing only the elements from the object passed into select which return true when passed through the block. 
(1..10).select {|x| x % 4 == 0} #=> [4, 8]
(1..10).select {|x| x.even? } #=> [2, 4, 6, 8, 10]
Sort Returns an array containing items in the original object sorted either according to the results of their <=> method or using the results of a block. The block should return -1, 0, or 1 depending on the comparison between a and b. #sort_by sorts a set of keys generated by mapping the values of the object through a block. 
["a", "c", "b"].sort #=> ["a", "b", "c"]
["a", "c", "b"].sort{|x, y| y <=> x } #=> ["c", "b", "a"]
["apples", "blackberries", "cheeses"].sort_by {|word| word.length}
#=> ["apples", "cheeses", "blackberries"]
